### What is it?
<b>GoBarber's Backend</b>

### Technologies used:
- [Node](https://nodejs.org/en/)

### How to run?
This project uses NodeJS, in order for it to work, frist navigate insde the project folder and run npm install to download the proper dependencies:
``` bash
cd gobarber-backend
npm install
npm run start
```
